# NestJS

## Instalación

```bash
# Instalamos los paquetes node_moudles
$ yarn install
```

## Ejecutar la aplicación

```bash
# watch developer
$ yarn run start:dev
```

```bash
# watch production
$ yarn run start:dev
```

## Estrutura del env

```bash
NAME_PROYECT=#
PORT_BASE_DATOS=#
BASE_DATOS=#
HOST=#
USER_DB=#
PASSWORD_DB=#
PORT_APLICATION=#
SYNCHRONIZE=#
```

## Comandos de apoyo

```bash
# genera modulo
$ nest g resource api/[nombre del modulo]

```
